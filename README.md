# AI Resume Analyzer

AI Resume Analyzer is an NLP-based application that analyzes resumes and compares them with job descriptions to compute match scores.

# Features
- Resume parsing and text extraction
- Job description comparison
- Resume-job match score calculation
- Skill gap detection for ATS optimization

# Tech Stack
- Python
- NLP
- Scikit-learn
- Flask
- Streamlit
